## ⚙️ Cᴏɴꜰɪɢᴜʀᴀᴛɪᴏɴ

### 🔴 Required Variables

| Variable | Description |
|---|---|
| `BOT_TOKEN` | Get from [@BotFather](https://t.me/BotFather) |
| `API_ID` | From [my.telegram.org](https://my.telegram.org) |
| `API_HASH` | From [my.telegram.org](https://my.telegram.org) |
| `OWNER_ID` | Your Telegram User ID |
| `MONGO_URL` | MongoDB Atlas URI from [cloud.mongodb.com](https://cloud.mongodb.com) |
| `GROUP_ID` | Main bot group ID |
| `CHARA_CHANNEL_ID` | Character log channel ID |

### 🟡 Optional Variables

| Variable | Default | Description |
|---|---|---|
| `BOT_USERNAME` | `Senpai_Waifu_Grabbing_Bot` | Bot username without @ |
| `SUPPORT_CHAT` | `THE_DRAGON_SUPPORT` | Support group username |
| `UPDATE_CHAT` | `Senpai_Updates` | Updates channel username |
| `IMGBB_API_KEY` | `None` | ImgBB API key for image uploads |

## 🚀 Dᴇᴘʟᴏʏ

<details>
<summary>📌 Deploy to VPS (Recommended)</summary>

```bash
# Step 1 — Install dependencies
apt update && apt install -y python3 python3-pip git tmux

# Step 2 — Clone the repo
git clone https://github.com/SenpaiLabs/SenpaiXWaifu.git
cd SenpaiXWaifu

# Step 3 — Configure environment
cp .env.example .env
nano .env   # Fill your credentials

# Step 4 — Install Python packages
pip3 install -U -r requirements.txt

# Step 5 — Run in background
tmux
python3 -m senpai
# Press CTRL+B then D to detach
```

</details>

<details>
<summary>📌 Deploy with Docker</summary>

```bash
# Build the image
docker build -t senpai-waifu-bot .

# Run with env file
docker run --env-file .env senpai-waifu-bot
```

</details>

```

start          — Start the bot & view help
guess          — Guess the spawned character
collection     — View your harem collection
shop           — Open the character shop
smode          — Change harem sorting mode
leaderboard    — View all leaderboards
bal            — Check your coin balance
fav            — Set favorite character
gift           — Gift a character to someone
trade          — Trade characters with a user
pay            — Send coins to another user
s              — View character from waifu ID
find           — Find characters by name
redeem         — Redeem codes for rewards
sclaim         — Claim your daily waifu

upload         — Add a new character
delete         — Remove a character
update         — Update character info
give           — Give character to a user
gen            — Generate coin redeem code
sgen           — Generate character redeem code
stats          — View database statistics
resetshop      — Reset a user's shop
set_on         — Enable rarity in group
set_off        — Disable rarity in group
lock           — Lock a character from spawning
unlock         — Unlock a character

ping           — Check bot latency
broadcast      — Message all users/groups
changetime     — Change spawn frequency globally
ctime          — Change spawn frequency per group
backup         — Create database backup
true           — Pull updates & restart bot
eval           — Execute Python code
exec           — Execute shell commands
```
